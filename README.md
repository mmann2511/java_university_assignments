# Java OOP Coursework

This repository contains a sequence of Java modules completed as part of my university coursework.
Each module builds toward foundational skills in:

- basic Java syntax and output
- user input and branching
- object-oriented programming (encapsulation, inheritance, polymorphism)
- domain object modeling
- arrays, aggregation, and CRUD-style list management
- file ingestion and CSV parsing
- exception handling
- unit testing using JUnit
